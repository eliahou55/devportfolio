Projets realisés

Meteor App

Le projet de stage consistait à développer une application réactive avec Meteor, offrant un accès client et administrateur. Les utilisateurs pouvaient créer des comptes individuels. L'objectif était de manipuler des collections de données via MongoDB. L'application était liée à deux API distinctes stockant des factures hiérarchisées, triées par date avec JavaScript et Node.js. En résumé, ce stage visait à créer une application réactive utilisant Meteor, permettant aux utilisateurs de gérer et trier des factures via des API connectées à MongoDB

Connexion,trie et selection d'API

Dans ce projet, l'objectif était d'implémenter un script JavaScript exécuté par Node.js pour réaliser une opération de filtrage de factures. Le code devait permettre la sélection d'un sous-ensemble de factures filles en se basant sur une date spécifiée pour la facture mère, avec cette dernière étant choisie par l'utilisateur. Ce processus utilise des fonctionnalités avancées de JavaScript, telles que la manipulation d'objets, les opérations de comparaison de dates et la gestion des tableaux de données. Le code implémenté assure également la gestion des erreurs et des cas limites pour garantir un fonctionnement fluide et robuste de l'application.
