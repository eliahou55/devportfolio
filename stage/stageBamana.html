Projets réalisés

Article Editor

Cette application web, développée avec Flask, offre une interface conviviale permettant aux utilisateurs de créer un compte, rédiger, éditer et publier des articles. L'accent est mis sur la simplicité et la clarté avec un tableau de bord intuitif. Les fonctionnalités CRUD (Create, Read, Update, Delete) sont au cœur du projet, permettant aux utilisateurs de gérer facilement leur contenu. La sécurité est renforcée avec l'authentification utilisateur. Chaque utilisateur dispose d'un profil personnalisable, et les notifications par e-mail sont incluses pour une expérience utilisateur complète.

Le projet détaillé

PyScrapeWeb : Générateur HTML d'Articles

Le cœur de ce projet vise à créer un script Python pour extraire des données d'articles de site mediatique. Les informations clés, telles que la catégorie, le titre, le contenu et l'image, sont utilisées pour générer des fichiers HTML organisés par catégorie. En intégrant cela avec Flask, le script ajuste également un fichier temp_urls.py pour créer des routes dynamiques basées sur les titres des articles. En résumé, le projet automatise le processus de récupération et d'affichage d'articles de manière structurée et élégante.
